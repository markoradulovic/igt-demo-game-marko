# Audio assets

Placeholder sound effects used by Phase 9's audio layer. Each file is a short
synthesized tone/sweep/click generated by `scripts/generate-placeholder-audio.mjs`,
NOT real production audio — they exist to verify the audio wiring end-to-end
without shipping any third-party binaries in the repo. Replace each with a CC0
sourced effect before delivery (e.g. freesound.org, kenney.nl).

| File               | Role             | Placeholder description            |
| ------------------ | ---------------- | ---------------------------------- |
| `spin-start.wav`   | Reel spin start  | Rising noise whoosh (~0.35s)       |
| `reel-land.wav`    | Per-reel land    | Low thud + noise click (~0.15s)    |
| `win.wav`          | Win jingle       | Ascending C-major arpeggio (~0.7s) |
| `click.wav`        | UI button click  | Short noise blip (~0.05s)          |
| `anticipation.wav` | Anticipation cue | Ascending sine sweep (~0.6s)       |

Regenerate placeholders with:

```bash
node scripts/generate-placeholder-audio.mjs
```
